# statics

C++ header only library inspired by the book "Engineering Mechanics STATICS" by R. C. Hibbeler.

Compile:

'''bash
cd C:\brg\2_code\statics
mkdir build
cd build
cmake -G "Visual Studio 17 2022" ..
cmake --build . --config Release
.\Release\MyProject.exe
'''
